#include <stdio.h>
#include <string.h>

int main(void) {
    const char *a = "casa";
    const char *b = "caso";

    printf("strcmp: %d\n", strcmp(a, b));
    printf("strncmp(3): %d\n", strncmp(a, b, 3));
    return 0;
}

Errores frecuentes

  • No validar entradas o retornos de funciones estándar.
  • No controlar casos borde (buffers, límites, punteros nulos).

Aplicación práctica

strcmp y strncmp se usan en comparación de claves, ordenación de cadenas y validación de contraseñas en aplicaciones C.
