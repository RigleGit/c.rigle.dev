Lo que aprenderás

Fundamentos sólidos de C

Variables, tipos de datos y operadores para entender cómo trabaja C desde la base.

Control de flujo y lógica

Condicionales y bucles (`if`, `switch`, `for`, `while`) para resolver problemas reales paso a paso.

Funciones y recursividad

Descomposición de problemas, reutilización de código y técnicas recursivas con criterio.

Punteros y memoria dinámica

Gestión de direcciones, `malloc/free` y buenas prácticas para evitar errores comunes.

Arrays, cadenas y estructuras

Modelado de datos con arrays, strings y `struct` para construir soluciones más completas.

Ficheros y entrada/salida

Lectura y escritura en archivos, además de manejo de recursos para programas robustos.

No es solo teoría: es práctica real

Cada ejercicio está diseñado para que escribas código desde el primer día. No solo lees, programas y aprendes haciendo.
